Smart Fish Finder AI
An AI-powered fishing probability predictor and safety dashboard for the Sri Lankan fishing community.

How it Works:
Machine Learning: Uses a Random Forest Classifier to predict fishing conditions.

Real-time Data: Fetches live weather updates via API.

Safety Features: Integrated Fuel Pass and offline GPS tracking concepts.
